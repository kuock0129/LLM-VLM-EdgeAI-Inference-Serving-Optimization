# LLM-VLM-EdgeAI-Inference-Serving-Optimization

This repository focuses on optimizing and serving state-of-the-art Large Language Models (LLMs) and Vision-Language Models (VLMs) for local inference on resource-constrained edge devices like the NVIDIA Jetson Orin Nano.

## Project Roadmap (6 Weeks)
- **Week 1**: Setup, Environment Fixing, & 4-bit Quantization Baseline
- **Week 2**: Post-Training Quantization (PTQ) Evaluation (INT8/INT4)
- **Week 3**: JAX/XLA Compiler-Level Optimization (Linear + GeLU Layer Fusion)
- **Week 4**: Multi-Model Serving via NVIDIA Triton Inference Server
- **Week 5 & 6**: Benchmarking, Profiling, & Final Deployment

## Hardware Environment
- Device: NVIDIA Jetson Orin Nano Developer Kit (8GB Unified Memory)
- OS/Platform: JetPack 6.x (Ubuntu 22.04 LTS)
